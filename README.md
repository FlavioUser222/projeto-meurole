<h1>Sobre o projeto</h1>
<p>Projeto desenvolvido no SENAC ,tem como objeto listar lugares/roles para conhecer em Patos de Minas</p>
<br/>

<h1>Funcionalidades</h1>
<p>API desenvolvida em NODE JS para listar,criar e excluir localidades adicionadas em um banco de dados NEON</p>
<br/>

<h1>Como executar o projeto</h1>
<p>Executar npm i na pasta-do-projeto,para instalar dependências</p>
<p>Nome do arquivo: ./index.js</p>
<p>Criar arquivo .env com váriavel DATABASE_URL</p>
<p>Colar a chave do banco NEON na váriavel DATABASE_URL</p>
<p>Iniciar projeto node >nome-do-arquivo</p>

<br/>

<h1>Pré Requisitos</h1>
<p>Ter NODEJS,VSCODE instalado em sua maquina</p>
